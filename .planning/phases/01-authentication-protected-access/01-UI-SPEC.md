## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline status spacing |
| sm | 8px | Compact form spacing, helper/error text separation |
| md | 16px | Default field, button, and card spacing |
| lg | 24px | Form-group spacing, auth card padding rhythm |
| xl | 32px | Auth page section gaps, card padding on desktop |
| 2xl | 48px | Separation between auth shell regions |
| 3xl | 64px | Page-level breathing room on desktop |

Exceptions: minimum interactive target size 44px for input fields, password toggle, and logout control

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--color-base-100` / `oklch(100% 0 0)` | Login page background surfaces, form surface, protected page canvas |
| Secondary (30%) | `--color-base-200` / `oklch(93% 0 0)` | Sidebar, auth card accents, input backgrounds, muted sections |
| Accent (10%) | `--color-primary` / `oklch(58% 0.158 241.966)` | Primary CTA, focused field ring, active nav state, loading emphasis |
| Destructive | `--color-error` / `oklch(70% 0.191 22.216)` | Invalid credentials feedback, session failure alerts, logout/error destructive messaging only |

Accent reserved for: Entrar button, authenticated-route active state, focused input outline, auth bootstrap loading indicator, success redirect emphasis after login. Do not use accent for every secondary link or neutral action.

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Entrar na conta |
| Empty state heading | Você ainda não iniciou sessão |
| Empty state body | Faça login com seu e-mail ou nome de usuário e senha para acessar o sistema. |
| Error state | Não foi possível entrar. Verifique seu e-mail ou nome de usuário e senha e tente novamente. Se o problema continuar, fale com o administrador. |
| Destructive confirmation | Encerrar sessão: no modal. Ação direta no clique, com toast `Sessão encerrada com sucesso.` e redirecionamento imediato para `/login`. |
